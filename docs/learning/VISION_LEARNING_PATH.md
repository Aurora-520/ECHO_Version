# 电赛视觉学习路线

面向已有电控经验、视觉经验较少的开发者。学习顺序与工程阶段一致。

1. 图像基础：像素、通道、灰度、曝光、增益、噪声、ROI 和坐标系。
2. 鲁棒性基础：光照、曝光、色温、反光、模糊、遮挡、负样本、安全拒绝和恢复。
3. 二值化与形态学：固定阈值、Otsu、自适应阈值、腐蚀、膨胀、开闭运算。
4. 轮廓与几何：面积、周长、矩、外接矩形/圆、圆形度、角点和直线。
5. 标定：内参、畸变、外参、单应性和像素到场地坐标。
6. 多帧状态：关联、滤波、跟踪、结果年龄和遮挡恢复。
7. 数据工程：原图、标注、失败帧、场景切片、训练/验证/测试划分、版本和哈希。
8. 神经网络：只在传统方法不足且数据足够时使用，并保留几何精修和置信度门禁。
9. 系统工程：非阻塞采集、协议、故障状态、回放、性能指标和赛场排障。

第一批练习应使用固定图片完成圆、矩形、中心和黑线提取，不直接从实时相机和云台控制开始。
